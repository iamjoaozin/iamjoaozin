# NexJob - Project Documentation

## Project Overview
NexJob is a modern freelancer marketplace connecting clients with freelancers.
Key features include:
-   Job posting and searching
-   Proposal submission
-   Secure payments via Pix
-   Premium subscription for freelancers

## Tech Stack
-   **Framework**: SvelteKit
-   **Language**: TypeScript
-   **Styling**: Tailwind CSS v4
-   **State Management**: Svelte 5 Runes
-   **Icons**: Lucide Svelte

## Architecture Plan

### Navigation Structure
-   **Public**: Home, Login, Register, Job Search, Premium Info
-   **Client**: Dashboard, Post Job, My Jobs, Profile
-   **Freelancer**: Dashboard, Find Work, My Proposals, Profile
